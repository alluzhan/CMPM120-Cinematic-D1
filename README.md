# CMPM80J-Cinematic-D1

Cinematic Requirements
----------------------------
3+ scenes: My D1 Cinematic has a total of 6 scenes which are, Preload, Intro, Outside, Inside, Menu, and Credits.
2+ audio assets: The cinematic uses a total of four audios, the bell and pop sound are used with my logo animation in
in the intro scene. The rain sounds are played in the outside scene, and the cafe music is played throughout the inside
and menu scene
2+ image assets: I used an image assets for the outside and inside scenes by other artists mentioned in the credits. The
intro and menu scenes use assets that I created for the logo and menu book. I used procreate to draw the assets out and I
used the procreate animation tool to create the animation pngs.
2+ geometric shapes: The intro scene utilizes a rectangle and lines to create a window behind the logo. Additionally, ellipse
and rectangular shaped are used in the outside, inside, and menu scenes for text and interactive buttons.
1+ logotype: The logotype text I created was my own logo in the intro scene. I used procreate text and drawing tools to make it
and used their built in animation tool.
1+ multi-line text: There is multi-line text in the outside and inside scene, which are used to guide the players towards a story,
and the menu buttons.
1+ fading animation: Between each scene there is a fading transition.
1+ object motion animation: In the outside scene there is an enter the cafe button which has a breathing tween effect. This makes it
so that it is emphasized to players that they should click on it.
